# 程序是关于rust语言的学习demo

#### 一、写在前面：
>1.使用本demo程序集必须已经安装rust；<br>
2.程序集所使用的rust版本：1.25.0；

#### 二、使用方式：
>1.克隆或下载本程序包至本地<br/>
2.进入rustDemo根目录，运行:
`cargo run `，
根据命令窗口的提示输入要查看的demo实例运行结果。


#### 三、目前的demo
##### 1.trait(特性)基本<br>
>源码路径: `src/about_trait/mytrait.rs`<br>
命令窗口指令: `1`

##### 2.trait在其他模块中实现<br>
>源码路径: `src/about_trait/custom.rs + mytrait.rs`<br>
命令窗口指令: `2`
